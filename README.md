Project Overview

The goal of this project is to create an interactive and user-
friendly restaurant website that allows customers to view

and order dishes through a digital menu. This web
application simulates the basic functionality of a
restaurant's online ordering system, with smooth
navigation and a dynamic interface.

Website Features
Introduction Page
• The homepage welcomes users with an introduction
to our restaurant, highlighting the vision behind our
food.
• The names of the students who worked on the project
are displayed here.
• A “Take a look at our restaurant” button leads users
to the interactive menu page.
Menu Page
• Displays a variety of dishes served at our restaurant.
• Dishes are organized in a clear and aesthetic layout.
• Each dish is represented with:
o An image
o Dish name
o Price

o “Add to Cart” button

Dish Details
• Users can click on any dish to open a pop-up or a new
section with:
o A detailed description of the dish
o Calorie information

Cart Functionality
• As users add dishes, the cart dynamically updates
with selected items.
• The cart displays:
o List of selected dishes
o Quantity
o Total price

How We Built the Website
• HTML (6 files):
o index.html – Introduction page
o menu.html – Menu listing
o details.html – Dish description pages for each
dish
• CSS (1 file):

o style.css – Contains all styling, including layout,
fonts, colours, hover effects, and responsive
design

• JavaScript (1 file):
o script.js – Manages:
▪ Navigation between pages
▪ Cart operations (add/remove items, total
calculation)
▪ Display of dish details

Challenges We Faced
• Designing a smooth and intuitive user interface
• Managing cart data
• Making the design responsive
• Ensuring clean code organization and page
transitions
